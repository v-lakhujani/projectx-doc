Welcome to projectx's documentation!
====================================

Projectx is a command-line based and easy to use tool


Features
########


- Download files associated to any sample-id to your local machine
- Can choose as where to download files
- Can see analysis list with flowcell, lims, workflow-name, panel-name and more
- Can see files available with any sample-id

Check out the :doc:`usage` section for further information, including how to :ref:`install <installation>` projectx and :ref:`play <play>`.
